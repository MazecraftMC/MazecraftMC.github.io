üí∞ MazePay

Lightweight economy plugin with dual account system, bank interest, and comprehensive API

GitHub Poggit

‚ú® Features

  • Dual Account System - Separate wallet and bank accounts
  • Bank Interest - Configurable interest rate on deposits
  • Custom Forms - Beautiful UI with customizable images
  • Multi-Database - SQLite, MySQL, JSON support
  • High Performance - Built-in caching and async operations
  • Developer API - Complete API with cancellable events
  • Fully Customizable - Messages, forms, economy settings

üì¶ Installation

  1. Download the latest release from GitHub Releases or Poggit
  2. Place the .phar file in your plugins/ folder
  3. Restart your server
  4. Customize plugin_data/MazePay/config.yml and messages.yml

üìã Commands

/balance [player]

View your or another player's balance

/pay <player> <amount>

Transfer money to another player

/bank

Open the interactive bank menu

/deposit <amount>

Deposit money from wallet to bank

/withdraw <amount>

Withdraw money from bank to wallet

/rich [10|20|30]

View top richest players

‚öôÔ∏è Configuration

economy:
  currency-symbol: "$"
  starting-balance:
    wallet: 1000
    bank: 0
  interest-rate: 0.5
  minimum-transaction: 1
  max-wallet: -1  # -1 = unlimited
  max-bank: -1

üîå API Quick Start

use PixelMCN\MazePay\MazePay;

$economy = MazePay::getInstance()->getEconomyManager();

// Get account
$account = $economy->getAccount("PlayerName");
$wallet = $account->getWallet();
$bank = $account->getBank();

// Add/Remove money
$economy->addMoney("PlayerName", 100.0, "wallet");
$economy->removeMoney("PlayerName", 50.0, "bank");

// Transfer between players
$economy->transfer("Sender", "Receiver", 100.0);

üõí MazeShop

Advanced shop & auction system with multi-level categories and native Bedrock UI

‚ú® Features

üè™ Shop System

  • Multi-level category structure
  • Native Bedrock UI (ChestUI)
  • Buy & sell items

‚öñÔ∏è Auction House

  • Player-to-player trading
  • Real-time bidding system
  • Auto-refund for outbid players
  • Configurable fees & limits
  • Optional (can be disabled)

üì¶ Installation & Requirements

‚ö†Ô∏è Required: MazePay or BedrockEconomy plugin

  1. Download from GitHub or Poggit
  2. Install either MazePay or BedrockEconomy
  3. Place MazeShop.phar in plugins/ folder
  4. Restart server
  5. Configure config.yml, shop.yml, and messages.yml

üìã Commands

/shop

Open the shop menu

/shop <category>

Open specific category

/sell [amount|all]

Quick sell items

/auction

Open auction house

/auction create <bid> <duration>

Create new auction

/auction bid <id> <amount>

Place a bid

‚öôÔ∏è Configuration Example

  max-auctions-per-player: 5

  items-per-page: 45
  allow-custom-blocks: true

üë• Parties & Friends

Modern party and friend system with multi-database support and cross-server synchronization

GitHub Poggit

‚ú® Features

üéâ Party System

  • Create and manage parties
  • Configurable member limits
  • Invite/remove players
  • Dedicated party chat
  • Leadership transfer
  • Cross-server persistence

ü§ù Friend System

  • Send/manage friend requests
  • Online/offline status tracking
  • Cross-server synchronization
  • Maximum friends limit
  • Private messaging
  • Request expiration

üì¶ Installation

  1. Download from GitHub or Poggit
  2. Place the .phar file in plugins/ folder
  3. Restart your server
  4. Configure plugins/PartiesAndFriends/config.yml
  5. Customize messages in messages.yml

üìã Commands

/party

Open party menu

/party invite <player>

Invite a player to party

/party kick <player>

Kick a member (leader only)

/friend

Open friends menu

/friend add <player>

Send friend request

/friend list

View friends list
